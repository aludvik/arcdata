- The loot guide is a tool that helps users understand the relationships between items.
- The loot guide is displayed on the bottom of the app in a separate section that splits the
  screen horizontally. 
- The loot guide takes as input a selection of items that has been made by the user in the
  items spreadsheet.
- The loot guide is only displayed when the user has selected one or more items in the main
  spreadsheet part of the app.
- The loot guide can be minized by the user by clicking a button.
- The main feature of the loot guide is the crafting guide.
- The crafting guide displays a directed graph of all the items selected, all the items that
  can be used to craft the selected items, and all the items that can be used to craft the
  crafting items, and so on recursively until only non-craftable items are left.
- The crafting guide also shows, on the edge from item to item, the number of input items
  needed to craft the output item.
- In addition to the crafting guide, the loot guide can display similar graphs showing what
  the selected items can be salvaged into or recycled into.

- There are two visualizations I want to show
  - A separate table displaying all the items in the graph where each entry in the table shows:
    - The item name, 
    - A list of what it's used to build in the selected items
    - A sum of all the weights going into that node in the graph
    - Items sorted from ascending from smallest weight to largest
  - A visualization of the graph itself, that shows:
    - An image of the item for each node
    - All edges to that node, with input quantities (weights) on the edges
    - Nodes sorted ascending from left/right or top/bottom according to the number of incoming edges

Cascade weights
- The weight calculation on nodes isn't quite right.
- Weights should cascade, if you need multiple of an intermediary resource, 
  then each instance of that resource should add weight to the downstream resources.
- For example, if you need 4 mechanical components, they take 7 metal and 4 rubber, 
  so the mechanical components should add 28 to metal's weight and 16 to rubber's weight

Scrub salvage inputs
- Some items have their recipe listed as another item that can be salvaged into it.
- This will create cycles and also is misleading, as crafting and recycling/salvaging should be treated separately.
